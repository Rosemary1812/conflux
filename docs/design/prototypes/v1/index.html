Conflux V1 原型

这一版先按你的标注修“页面成分和入口位置”，不先锁最终视觉风格。

  • 单聊主页面 修正 logo、会话分组、底部用户设置入口、上传型输入框、右栏可收起 / 拖拽
  • 新建单聊 直接进入空白对话；首条消息里只允许 @ 一个 Agent，确认后锁定
  • 新建群聊 直接进入空白群聊；首条消息可 @ 多个 Agent，Orchestrator 自动加入
  • 群聊静态预览 更明确地区分单聊 / 群聊身份、任务卡片、Orchestrator 占位头像、禁用发送
  • 设置弹层预览 不再独立跳转；预览从用户头像弹出的设置层，含 Provider / Skill / 自建 Agent
  • 空状态与错误 无会话、healthcheck 失败、连接中、发送失败等状态页
