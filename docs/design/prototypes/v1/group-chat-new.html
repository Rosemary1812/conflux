新建群聊

群聊会话
群聊进入后不预选 Agent，而是在首条消息中通过多个 @Agent 完成成员选择；只要你开始 @ 多个 Agent，Orchestrator 就会自动加入。
第一步：在输入框里连续输入多个 Agent mention，例如 @claude-code @codex。
第二步：系统自动补上 @orchestrator，用于分派任务、聚合反馈与管理上下文。
第三步：发送后形成群聊成员列表，后续消息默认沿用这组成员。
@claude-code @codex @orchestrator 自动加入
示例：@claude-code @codex 帮我并行做设置页 UI 和接口校验
规则：群聊允许多 Agent；Orchestrator 不手动选择，只要有多 Agent 群聊就默认存在。
